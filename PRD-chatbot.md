# PRD: Health Companion Chatbot - 个人/群组聊天 Agent

> 版本: v0.1 | 日期: 2026-03-05 | 作者: Jamie (康金宇) & Bailey (吴小北)

## 一、产品概述

### 一句话描述

Health Companion 是面向新加坡糖尿病患者的多模态聊天 Agent，支持文字、语音、图片输入，通过意图分类 + 情绪感知路由到专业 Agent，提供个性化的健康管理对话体验。

### 背景与动机

- **所属项目**: SG INNOVATION Challenge - AI 远程健康管理平台
- **Chatbot 的角色**: 平台的核心交互入口。用户所有的健康管理行为（饮食记录、用药咨询、情绪倾诉、指标追踪）都通过 chatbot 进行
- **当前状态**: Bailey 已完成文字 + 语音对话的基础架构。Jamie 已完成 Vision Agent（图片识别模块）。下一步是集成 + 完善

### 目标用户

- **主要用户**: 新加坡糖尿病患者（核心为 2 型糖尿病）
- **使用场景**:
  - 日常对话：问健康问题、闲聊、情绪倾诉
  - 图片交互：拍食物/药物/化验单，获取识别结果 + 专业建议
  - 语音交互：语音输入 + 声学情绪识别
  - 任务打卡：血糖记录、饮食打卡、运动提醒

## 二、核心功能（MVP 范围）

### 必须有（Must Have）

1. **意图分类 + 路由**: 识别用户意图（健康咨询/情绪倾诉/任务/预警/闲聊），路由到对应 Agent
2. **情绪感知**: 文字 + 语音双通道识别情绪，影响回复策略（先安抚还是先推进）
3. **专家 Agent 多轮追问**: 逐步收集血糖、饮食、用药信息，最后给综合建议
4. **陪伴 Agent**: 情绪支持 + 心理危机检测（自动触发预警 + 危机热线）
5. **图片识别集成**: 用户发图片 → Vision Agent 识别 → 结果供专家 Agent 使用（减少追问轮数）
6. **数据持久化**: 用户档案 + 对话历史的数据库存储，支持长短期记忆
7. **下游触发器**: 情绪日志、任务请求、预警信号输出给其他模块（Julia 预警、Chayi 任务）
8. **限流降级**: 主模型不可用时自动切换备用模型

### 最好有（Nice to Have）

1. **群组聊天**: 多人健康管理场景
2. **追问链退出优化**: 用户想换话题时能更智能地检测

### 明确不做（Out of Scope）

- **不做前端 UI**: 后端逻辑只提供 API / CLI，前端由其他同学负责
- **不做多语言扩展**: MVP 只支持中文 + 英文

## 三、用户流程

### 核心用户旅程

```
用户输入（文字 / 语音 / 图片）
  → [语音] 转文字 + 提取声学情绪
  → [图片] Vision Agent 识别 → 结构化数据
  → 意图分类 + 情绪识别
  → 策略决策（意图 + 情绪 → 回复策略）
  → 路由到对应 Agent → 生成回复
  → 同时生成下游触发器（预警/任务）
```

### 图片交互流程

```
用户发送图片
  → Vision Agent 识别场景（食物/药物/化验单/未知）
  → 返回结构化数据
  → 专家 Agent 利用识别结果，跳过已有信息的追问，直接给建议
  → 如果无法识别，告知用户
```

## 四、产品设计要点

### 核心机制

- **意图 + 情绪双轴路由**: 意图决定去哪个 Agent，情绪决定 Agent 怎么说话
- **多轮追问**: 专家 Agent 逐步收集信息再给综合建议，而非一问一答
- **图片加速追问**: 图片识别结果可以跳过对应的追问步骤（比如拍了食物照片就不用再问"吃了什么"）
- **危机兜底**: 任何路径下检测到自杀/自伤意图，立即触发危机响应

### 设计原则

- **策略可控可审计**: 医疗场景下策略必须是明确规则，不能让 LLM 自行决策
- **情绪优先**: 用户情绪不好时，先安抚再推进任务
- **模块解耦**: Vision Agent 无状态、chatbot 有状态，通过 JSON 接口连接

## 五、技术方向

| 层面 | 选型 | 理由 |
|------|------|------|
| 编排框架 | LangGraph | 状态图 + 条件路由 |
| 对话模型 | SEA-LION 32B | 主办方推荐 |
| 推理模型 | SEA-LION 70B | 专家 Agent 深度推理 |
| 视觉模型 | Gemini 2.5 Flash | 图片识别 |
| 语音处理 | MERaLiON | 语音转文字 + 情绪识别 |
| 降级备用 | Cloudflare Gemma | 限流时自动切换 |
| 数据持久化 | 待定 | brainstorm 阶段确定方案 |
