# Task Page Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an interactive Task page with stacked cards, expand/collapse animation, task completion with points, and stats tracking — all pure frontend with mock data.

**Architecture:** Single-file page component (`app/task/page.js`) using React `useState` for all state. Task data is hardcoded. Modals for photo capture and body check-in are inline JSX. Reuses existing `TopBar` component.

**Tech Stack:** Next.js 14 (App Router), Tailwind CSS, React useState, CSS transitions

```jsx
const TASKS = [

    color: "#a8cce8", description: "Small notes today, better insights tomorrow.",

  const dailyPts = completedCount * PTS_PER_TASK;
  const plantPts = completedCount * PTS_PER_TASK;

      <div className="flex-1 overflow-y-auto pb-4">

      </div>
```

```jsx
{/* Card Stack */}
<div className="px-4 mt-2">
  {TASKS.map((task) => {
    const isExpanded = task.id === "sunset" || expandedId === task.id;

          </h3>
          {isCompleted ? (
            <span className="text-sm font-bold italic text-[#ff6b8a]">
              10 pt<br />earned!
            </span>
          ) : (
            <div className="w-8 h-8 rounded-full bg-white/60" />
          )}
        </div>

        {/* Expanded content */}
        <div
          style={{
            maxHeight: isExpanded ? 120 : 0,
            opacity: isExpanded ? 1 : 0,
            overflow: "hidden",
            transition: "max-height 300ms ease, opacity 300ms ease",
          }}
        >
          <p
            className="text-sm text-white/90 mt-2"
            style={{ textDecoration: isCompleted ? "line-through" : "none" }}
          >
            {task.description}
          </p>

          {task.extraInfo && (
            <p className="text-sm text-white mt-2">
              <span className="font-semibold">{task.extraInfo}</span>
            </p>
          )}

          {task.logType !== "none" && (
            <button
              disabled={isCompleted}
              onClick={(e) => {
                e.stopPropagation();
                handleLogClick(task);
              }}
              className="mt-3 px-6 py-2 rounded-full text-sm font-semibold text-white"
              style={{
                backgroundColor: isCompleted ? "#8bc34a" : `${task.color}dd`,
                filter: isCompleted ? "none" : "brightness(0.85)",
                opacity: isCompleted ? 1 : 1,
              }}
            >
              {isCompleted ? task.completedLabel : "Log Here"}
            </button>
          )}
        </div>
      </div>
    );
  })}
</div>
```

### Task 3: Add completion flow — photo modal and body form

```jsx
// Replace the stub from Task 2
const handleLogClick = (task) => {
  if (completedTasks.has(task.id)) return;
  setActiveTaskId(task.id);
  if (task.logType === "photo") {
    setShowPhotoModal(true);
  } else if (task.logType === "form") {
    setShowBodyModal(true);
  }
};

const completeTask = (taskId) => {
  setCompletedTasks(prev => new Set([...prev, taskId]));
};

const handleFileChange = (e) => {
  if (e.target.files?.[0]) {
    completeTask(activeTaskId);
    setShowPhotoModal(false);
    setActiveTaskId(null);
  }
  e.target.value = "";
};

const handleBodySubmit = () => {
  completeTask(activeTaskId);
  setShowBodyModal(false);
  setActiveTaskId(null);
  setBodyForm({ waist: "", weight: "" });
};
```
